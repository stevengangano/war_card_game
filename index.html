WAR Rules

The full name of this game is Casualty War.

Scoring:

Ace = 14

King = 13

Queen = 12

Jack = 11

All number cards equal their numbers

Play:

The deck of cards is split in half with both players getting 26 cards each. Each player lays one card face up. The player with the higher value takes all the cards.

War:

Occurs whenever there is a match in value one on each side. Each player places another three cards face down, then one card face up and proceed as in regular play, with the higher value taking all the cards. If another match occurs during war, the war rule applies until there is a winner.

Win:

Player to have all 52 cards wins.
